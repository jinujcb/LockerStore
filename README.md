Introduction: Locker Store is an application that helps to create, modify and delete records based on Locker's location, locker bank and locker number.

Steps used to create the project:
1. Created an Asp.NET MVC project in Visual Studio
2.Installed node.js and used npm commands to initialise and set up React folder in the project.
3.Created database named Locker containing 4 tables namely Location, LockerBanks, Lockers and LockerLists on Azure.
4.Used Db first approach in connecting to azure database in Model folder of the Locker_Store project.
5.Created an Interface folder containing interfaces for the classes Location, LockerBanks, Lockers and LockerLists.
6.The controller folder has the class with functions to fetch records from the database and present it to View.
7.Scripts folder contains the react folder to render and return the Table containing the list of locker records using React states.

Technologies used:
.Net C#, LINQ, MVC, SQL Server, HTML, JavaScript, React, Semantic UI React, Microsoft  Azure
